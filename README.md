# Python图书馆座位预约系统 python193

## 项目概述

本项目是基于Python的图书馆座位预约系统，它允许用户通过系统完成座位预约、自习室选择等功能，帮助图书馆有效管理座位资源。

## 技术栈

- 后端：Django
- 数据库：MySQL
- 前端：HTML, CSS, JavaScript

## 功能模块

- **用户模块**
  - 登录注册
- **预约管理**
  - 座位预约
  - 预约记录
- **用户行为**
  - 签到
- **自习室管理**
  - 自习室选择
  - 自习室管理
- **系统管理**
  - 用户管理
  - 预约管理
  - 警告管理
  - 提示管理

## 系统角色

- 普通用户：完成登录注册、预约座位、查看预约记录等操作。
- 管理员：管理用户信息、自习室配置、预约情况以及警告和提示信息。

## 运行环境

- Python环境
- MySQL数据库

## 部署步骤

1. 准备Python运行环境
2. 安装依赖库（Django, MySQL等）
3. 初始化数据库，执行SQL文件

5. 运行Django项目
